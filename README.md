# My Final Project
This is our final project for CSCI 120. Our Project is a Inventory Management System

Our project is an Inventory Management System designed to help users track and manage items in a store or warehouse. The program will allow users to add and remove items, update and monitor item quantities, search for specific products, and receive alerts when inventory levels are low. This system is meaningful because inventory tracking is a common real-world problem for businesses, and our program provides a simple and efficient way to organize and manage data.

The program will use advanced data structures such as dictionaries and lists to store and organize inventory data, as well as classes to represent items and manage the system. Key features will include adding new items with attributes (such as name, quantity, and category), removing items, searching for items by name, and automatically identifying low-stock items based on a defined threshold. Our team will ensure the program is user-friendly, well-documented with comments and docstrings, and easy to understand. Through this project, we aim to strengthen our understanding of object-oriented programming and data management while building a practical application.
